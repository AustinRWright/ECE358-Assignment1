Lab 1: M/M/1 and M/M/1/K Queue Simulation 
-------------------------------------------------
by Nathan Voskamp and Austin Wright

File List
-------------------------------------------------
DiscreteEventSimulator.py -- contains base DES class and implemented classes
event.py -- Base event type and the Arrival, Departure, and Observer events
q1.py -- generate 1000 exponential random variables
q3.py -- simulate the M/M/1 queue
q4.py -- simulate the M/M/1 queue with queue utilization of 1.2
q6.py -- simulate the M/M/1/K queue
report1.pdf -- lab report

Run simulations
-------------------------------------------------
To generate 1000 exponential random variables use:
python3 q1.py

To simulate the M/M/1 queue use:
python3 q3.py

To simulate the M/M/1 queue with queue utilization of 1.2 use:
python3 q4.py

To simulate the M/M/1/K queue use:
python3 q6.py
